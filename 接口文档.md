# Ariadne 接口文档（API v1）

## 1. 文档说明
- 基础路径：`/api/v1`
- 数据格式：`application/json`（文件上传除外）
- 编码：`UTF-8`
- 时间格式：`ISO-8601`（UTC）
- 幂等键：长任务接口建议传 `Idempotency-Key`

## 2. 通用约定

### 2.1 请求头
- `Content-Type: application/json`
- `X-Request-Id: <uuid>`（可选，客户端生成）
- `Idempotency-Key: <string>`（生成/导出类任务推荐）

### 2.2 统一响应结构
成功：
```json
{
  "code": 0,
  "message": "ok",
  "trace_id": "tr_01H...",
  "data": {}
}
```

失败：
```json
{
  "code": 10001,
  "message": "invalid topic length",
  "trace_id": "tr_01H...",
  "error": {
    "field": "topic",
    "reason": "length must be between 2 and 120"
  }
}
```

### 2.3 HTTP 状态码约定
- `200`：查询或动作成功
- `201`：资源创建成功
- `400`：参数错误
- `401`：未授权（如后续启用账号）
- `403`：禁止访问
- `404`：资源不存在
- `409`：版本冲突/重复提交冲突
- `413`：上传文件过大
- `415`：不支持的媒体类型
- `422`：业务校验失败
- `429`：请求过于频繁
- `500`：服务内部错误
- `503`：依赖服务不可用

## 3. 错误码定义
- `10001` 参数校验失败
- `10002` 非法状态转移
- `10003` 资源不存在
- `10004` 幂等冲突
- `10005` 版本冲突
- `10006` 文件格式不支持
- `10007` 文件大小超限
- `10008` 解析失败
- `10009` 检索无结果
- `10010` 生成失败
- `10011` 导出失败
- `10012` 离线模式禁止远程调用
- `10013` 权限不足

## 4. 枚举定义
- `difficulty`：`beginner | intermediate | advanced`
- `style`：`intuitive | rigorous | engineering`
- `template`：`tutorial | qa | project`
- `job_phase`：`queued | retrieving | generating | assembling | done | failed`
- `asset_status`：`queued | processing | ready | failed`
- `chat_mode`：`brief | deep`
- `credibility`：`high | medium | low`

## 5. 课件生成模块

## 5.1 创建生成任务
`POST /api/v1/coursewares/generate`

请求体：
```json
{
  "topic": "Transformer",
  "keywords": ["attention", "encoder", "decoder"],
  "difficulty": "beginner",
  "style": "engineering",
  "template": "tutorial",
  "retrieval": {
    "web_enabled": true,
    "source_weight": {
      "doc": 0.5,
      "blog": 0.2,
      "paper": 0.3
    },
    "domain_whitelist": ["arxiv.org"],
    "domain_blacklist": []
  },
  "profile_id": "pf_001"
}
```

校验规则：
- `topic` 长度 2~120
- `keywords` 最多 10 个
- 单个 keyword 长度 1~64
- `source_weight` 三项总和应为 1（允许后端自动归一化）

响应：
```json
{
  "code": 0,
  "message": "ok",
  "trace_id": "tr_xxx",
  "data": {
    "job_id": "job_001",
    "courseware_id": "cw_001",
    "phase": "queued"
  }
}
```

## 5.2 查询生成进度

响应：

## 6. Chunk 学习与问答模块

## 6.1 目录与内容
`GET /api/v1/coursewares/{courseware_id}/chunks`

查询参数：
- `include_content=true|false`
- `only_favorite=true|false`

## 6.2 更新 chunk 学习状态
`PATCH /api/v1/chunks/{chunk_id}/state`

请求体：
```json
{
  "understand_state": "understood",
  "is_favorite": true,
  "collapsed": false
}
```

## 6.3 提问（带上下文）
`POST /api/v1/chunks/{chunk_id}/ask`

请求体：
```json
{
  "question": "这个公式为什么要除以 sqrt(dk)？",
  "page_id": "pg_001",
  "chunk_id": "ck_001",
  "selection": {
    "text": "softmax(QK^T / sqrt(dk))",
    "start": 120,
    "end": 142
  },
  "mode": "deep"
}
```

响应：
```json
{
  "code": 0,
  "message": "ok",
  "trace_id": "tr_xxx",
  "data": {
    "answer_id": "ans_001",
    "answer": "...",
    "linked_chunk_id": "ck_001",
    "next_suggestions": ["要不要我画一个维度例子？"],
    "sources": [
      {
        "title": "Attention Is All You Need",
        "url": "https://arxiv.org/abs/1706.03762",
        "domain": "arxiv.org",
        "credibility": "high"
      }
    ]
  }
}
```

## 6.4 采纳回答补充到 chunk
`POST /api/v1/chunks/{chunk_id}/append`

请求体：
```json
{
  "answer_id": "ans_001",
  "action": "accept",
  "target": "current_chunk"
}
```

说明：
- `action=accept`：写入正文并创建新版本
- `action=reject`：不写正文，仅记录决策日志

## 6.5 chunk 来源列表
`GET /api/v1/chunks/{chunk_id}/sources`

响应字段：
- `title`
- `url`
- `domain`
- `published_at`
- `credibility`

## 7. AI 侧边栏会话模块

## 7.1 创建会话
`POST /api/v1/chat/sessions`

请求体：
```json
{
  "courseware_id": "cw_001",
  "page_id": "pg_001",
  "chunk_id": "ck_001"
}
```

## 7.2 发送消息
`POST /api/v1/chat/messages`

请求体：
```json
{
  "session_id": "cs_001",
  "message": "继续展开刚才的解释",
  "continue_from_message_id": "msg_009"
}
```

## 7.3 获取历史会话
`GET /api/v1/chat/sessions?courseware_id=cw_001&page_id=pg_001`

## 7.4 AI 改写草稿
`POST /api/v1/pages/{page_id}/rewrite-draft`

请求体：
```json
{
  "chunk_id": "ck_001",
  "instruction": "改成更直觉的讲解风格",
  "scope": "current_chunk"
}
```

响应：
```json
{
  "code": 0,
  "message": "ok",
  "trace_id": "tr_xxx",
  "data": {
    "draft_id": "dr_001",
    "original": "...",
    "rewritten": "..."
  }
}
```

## 7.5 应用草稿
`POST /api/v1/pages/{page_id}/apply-draft`

请求体：
```json
{
  "draft_id": "dr_001",
  "expected_version": 3
}
```

错误边界：
- `expected_version` 与当前版本不一致，返回 `409` + `10005`

## 7.6 撤销最近修改
`POST /api/v1/pages/{page_id}/undo`

请求体：
```json
{
  "expected_version": 4
}
```
